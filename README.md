# Description

## Dev environment

1. Clone the repository
2. Copy `.env.template` to `.env` and fill the variables.
3. Run `npm intall`.
4. Run database using `docker compose up -d`
5. Run migrations `npx primsa migrate dev`
6. Run seed `npm run seed`
7. Clean LS of the browser
8. Run `npm run dev`
